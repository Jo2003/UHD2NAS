UHD2NAS - Help

Overview

UHD2NAS re-encodes MKV files from Blu-ray or 4K Ultra HD Blu-ray discs into space-efficient HEVC 10-bit files suitable for NAS storage and streaming to media players.

The application automatically handles:

  • Crop detection – removes black bars
  • Dolby Vision conversion (Profile 7 → 8.1) for 4K UHD sources
  • HDR metadata preservation for 4K content
  • Audio and subtitles are copied without re-encoding

Main Window

Files

  • Source MKV – Select the original MKV file (ripped from Blu-ray via MakeMKV).
  • Target MKV – The output file path. Automatically suggested when you pick a source file. If the target already exists, you will be asked to confirm overwrite.

Encoder Settings

  • Encoder – Choose your encoding method:
    • Software (libx265) – CPU-based, best quality per bit, slowest.
    • Intel QuickSync – Hardware encoder using Intel integrated GPU.
    • NVIDIA NVEnc – Hardware encoder using NVIDIA GPU (RTX 20+).
    • AMD AMF – Hardware encoder using AMD GPU.
  • Quality – Slider from 18 (best quality, larger file) to 25 (lower quality, smaller file). Recommended: 20 for 4K, 22–23 for Full HD.

Progress

  • Step Progress – Shows which step of the workflow is currently active (e.g., crop detection, encoding, Dolby Vision injection).
  • Encode Progress – Shows the encoding progress as a percentage based on frames processed.

Buttons

  • Start – Begin the encoding process. Disabled while encoding is in progress.
  • Abort – Cancel the current operation. Temporary files are cleaned up.
  • Quit – Exit the application.

Log Output

The log area shows the raw output from ffmpeg, dovi_tool and mkvmerge, including the exact command lines being executed (prefixed with [CMD]). Useful for debugging and monitoring encode progress.

Settings

Access via the Settings... menu entry.

Tool Paths Tab

Configure the paths to the required external tools:

  • ffmpeg – The video encoder/decoder.
  • ffprobe – Used for analyzing input files (typically in the same folder as ffmpeg).
  • dovi_tool – Required only for 4K discs with Dolby Vision metadata.
  • mkvmerge – Used for the final muxing step (part of MKVToolNix).

Download links are provided if you need to obtain the tools.

Command Templates Tab

All tool commands are stored as editable templates with {placeholder} variables. Advanced users can modify these to adjust encoding parameters without recompiling.

Available placeholders:

PlaceholderDescription
{ffmpeg}Path to ffmpeg
{ffprobe}Path to ffprobe
{dovi_tool}Path to dovi_tool
{mkvmerge}Path to mkvmerge
{input}Source file path
{output}Target file path
{crop_w}Detected crop width
{crop_h}Detected crop height
{crop_x}Crop X offset
{crop_y}Crop Y offset
{crop_top}Pixels removed from top
{crop_bottom}Pixels removed from bottom
{crf}Quality value from slider
{rpu_file}Extracted RPU file path
{encoded_hevc}Intermediate encoded HEVC path
{injected_hevc}HEVC with injected RPU path
{fps}Detected source framerate (e.g. 23.976)

Click Reset to Defaults to restore all templates to their original values.

Workflow

Full HD Blu-ray (1080p, SDR)

  1. Crop detection (black bar removal)
  2. Encode to HEVC 10-bit with audio/subtitle copy

4K UHD Blu-ray without Dolby Vision

  1. Crop detection
  2. Encode to HEVC 10-bit with HDR metadata + audio/subtitle copy

4K UHD Blu-ray with Dolby Vision (Profile 7)

  1. Crop detection
  2. Extract RPU, convert Profile 7 → 8.1, and reset crop offsets (-m 2 -c)
  3. Encode video (HEVC 10-bit, video only)
  4. Inject RPU into encoded stream
  5. Final mux (combine video with original audio and subtitles)

Tips

  • Quality 20 is a good default for 4K content (close to transparent).
  • Quality 22–23 works well for Full HD, since compression artifacts are less visible at 1080p.
  • Hardware encoders are 5–10x faster than software but produce slightly larger files at equivalent quality.
  • NVEnc preset p7 gives the best quality from NVIDIA hardware at the cost of some speed.
  • Files with spaces in the name are fully supported.
  • The application detects whether the source is Full HD or 4K automatically based on the crop dimensions.

Requirements

  • ffmpeg (with HEVC encoder support)
  • ffprobe (included with ffmpeg)
  • dovi_tool (only needed for Dolby Vision content)
  • mkvmerge (part of MKVToolNix, needed for Dolby Vision final mux)
